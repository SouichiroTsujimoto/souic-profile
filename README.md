# souic-profile

個人ポートフォリオサイト

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Package Manager**: Bun
- **Styling**: CSS Modules + Tailwind CSS
- **Transitions**: next-view-transitions

## Getting Started

```bash
# 依存関係のインストール
bun install

# 開発サーバーの起動
bun dev
```

[http://localhost:3000](http://localhost:3000) をブラウザで開いてください。

## Scripts

```bash
bun dev      # 開発サーバー起動
bun build    # プロダクションビルド
bun lint     # リンター実行
bun format   # フォーマット実行
bun check    # リント & フォーマットチェック
```
